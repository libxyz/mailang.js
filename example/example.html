Mai语言帮助文档

内置函数

MA(data, period) - 移动平均线
计算数据的移动平均值
示例: MA5 := MA(C, 5)
SMA(data, period) - 简单移动平均线
MA函数的别名
示例: SMA5 := SMA(C, 5)
CROSS(a, b) - 金叉检测
检测a是否上穿b（前一期ab）
示例: CROSSOVER := CROSS(MA5, MA10)
CROSSDOWN(a, b) - 死叉检测
检测a是否下穿b（前一期a>b且当期a示例: CROSSDOWN := CROSSDOWN(MA5, MA10)
IFELSE(condition, true_value, false_value) - 条件选择
根据条件返回不同的值
示例: RESULT := IFELSE(C > O, 1, -1)
MAX(a, b, ...) - 最大值
返回多个值中的最大值
示例: MAX_PRICE := MAX(H, C)
MIN(a, b, ...) - 最小值
返回多个值中的最小值
示例: MIN_PRICE := MIN(L, C)
SUM(a, b, ...) - 求和
返回多个值的和
示例: TOTAL := SUM(C, O, H, L)
PRINT(...) - 打印输出
将值输出到控制台
示例: PRINT("当前价格:", C)

市场数据关键字

  • O - 开盘价 (Open)
  • H - 最高价 (High)
  • L - 最低价 (Low)
  • C - 收盘价 (Close)

运算符

  • 算术运算符: +, -, *, /
  • 比较运算符: >, <,>=, <=,=, <>
  • 逻辑运算符: &&, ||
  • 赋值运算符: := (普通赋值), : (显示赋值)

用户自定义参数

您可以在"用户自定义参数"区域定义全局变量，这些变量可以在Mai代码中直接使用。

示例:
在参数区域输入: 
SHORT_PERIOD=5
                            LONG_PERIOD=20
                            THRESHOLD=2.5

在代码中使用: 
MA_SHORT := MA(C, SHORT_PERIOD)
                            MA_LONG := MA(C, LONG_PERIOD)
                            SIGNAL := IFELSE(MA_SHORT > MA_LONG + THRESHOLD, 1, 0)

控制流

支持IF-THEN-ELSE条件语句:

IF 条件 THEN BEGIN
                        // 条件为真时执行的代码
                        END
                        ELSE BEGIN
                        // 条件为假时执行的代码
                        END
